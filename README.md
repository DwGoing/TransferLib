# transfer_lib

### 使用说明
> go get -u github.com/DwGoing/transfer_lib/pkg 

### 功能列表
|     **功能**     |   **接口名**   | **说明** |    **链支持**    |
| :--------------: | :------------: | :------: | :--------------: |
| **获取账户信息** |   GetAccount   |          | BTC/ETH/TRON/BSC |
|   **获取余额**   |   GetBalance   |          |   ETH/TRON/BSC   |
|     **转账**     |    Transfer    |          |   ETH/TRON/BSC   |
| **查询交易信息** | GetTransaction |          |   ETH/TRON/BSC   |
